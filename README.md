rooms-booking
==============================

Service with example REST API for room booking system.

## Prerequisites

Java 17

## Building

```./gradlew build```

## Running

```./gradlew bootRun```

## Running tests

* All tests (unit and integration)

```./gradlew test```

## Running SonarCloud Scan

* Run SonarCloud Scan

```./gradlew sonarqube -Dsonar.login=yourAuthenticationToken --info```
